HOW TO COMPILE ON OVERLEAF
===========================

1. Go to https://overleaf.com and create a new project
   → "Upload Project" → upload this entire ACR-QA-Book folder as a ZIP

2. Set the compiler:
   → Menu → Compiler → pdfLaTeX

3. Set the main document:
   → Menu → Main document → main.tex

4. Compile order (first time only):
   pdfLaTeX → BibTeX → pdfLaTeX → pdfLaTeX
   (Overleaf handles this automatically after the first compile)

5. Click "Recompile" — your PDF will appear on the right.
