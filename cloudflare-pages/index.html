How It Works

🔍

19 Engines

Bandit, Semgrep, Ruff, Vulture, Radon, gosec, staticcheck, ESLint, Trivy, jscpd + IaC scanner run in parallel.

🗂️

One Schema

All outputs normalised into CanonicalFinding. Cross-tool deduplication by deterministic fingerprint.

🤖

RAG-Grounded AI

Groq LLaMA-3.3-70B explains each finding, grounded in a 66-rule knowledge base. Semantic entropy (3× runs) detects hallucinations.

🎯

P4 Confirmed Tier

4-criterion gate: HIGH severity + 22-rule ConfirmedRuleSet + production code + Bandit HIGH confidence → 96.4% precision.

Get Started in 30 Seconds

# Pull and run (no config needed for a quick test)
docker run --rm \
  -v $(pwd):/scan \
  -e GROQ_API_KEY_1=your_key \
  ghcr.io/ahmed-145/acrqa:latest \
  python3 CORE/main.py --target-dir /scan --rich
# Full stack with dashboard
git clone https://github.com/ahmed-145/ACR-QA.git
cd ACR-QA && cp .env.example .env
docker compose up -d
# → open http://localhost:8000/docs
