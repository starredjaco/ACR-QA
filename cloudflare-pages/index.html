Exploit Verification

For every Confirmed finding, ACR-QA spins up an ephemeral Docker sandbox and fires real payloads. We don't report it unless we can show it firing.

Three-tier verdict: verified-exploitable · verified-unexploitable · unverified.

Get Started

Running in 30 seconds

# Option A — Docker (no clone needed)
docker run --rm \
  -v $(pwd):/scan \
  -e GROQ_API_KEY_1=gsk_your_key \
  ghcr.io/ahmed-145/acrqa:latest \
  python3 CORE/main.py --target-dir /scan --rich
# Option B — Full stack (API + dashboard + monitoring)
git clone https://github.com/ahmed-145/ACR-QA.git && cd ACR-QA
cp .env.example .env # add GROQ_API_KEY_1
docker compose up -d
# FastAPI: http://localhost:8000/docs | Grafana: http://localhost:3005
# Option C — GitHub Codespaces (zero local setup)
# Click "Open in GitHub Codespaces" in the repo README — takes ~2 min
